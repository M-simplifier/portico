# Contributing

External contributions are welcome.

Portico is still pre-beta, so normal OSS cadence is not guaranteed yet. Review, merge timing, release timing, and response speed may be slower than a mature public package.

Please keep contributions aligned with the current direction:

- published static sites, not app-shell behavior
- semantic site primitives over raw markup
- theme concerns separate from the content model
- `import Portico` as the public entry point

If a change starts moving Portico toward a different product category, stop and check whether it belongs elsewhere.
